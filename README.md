# Breast Cancer Classification using Machine Learning
**AI and ML | Current Trends and Topics in Computing | March 2026**

## Author
- Marcos, Carl Ernard M.

## About
This project applies supervised machine learning (Logistic Regression) to classify
breast tumors as malignant or benign using the Breast Cancer Wisconsin dataset.

**Result: 98.25% accuracy on test data**

## Dataset
Breast Cancer Wisconsin (Diagnostic) — built into scikit-learn  
569 samples | 30 features | 2 classes (malignant / benign)

## How to Run
1. Install dependencies:
   pip install scikit-learn matplotlib numpy
2. Run the script:
   python breast_cancer_classification.py
